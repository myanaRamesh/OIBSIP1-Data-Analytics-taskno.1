# Retail Sales Data Analysis

## 📌 Objective
To perform Exploratory Data Analysis (EDA) on retail sales dataset and extract meaningful insights.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## 📊 Steps Performed
1. Data Loading and Cleaning
2. Descriptive Statistics
3. Time Series Analysis
4. Customer and Product Analysis
5. Data Visualization

## 📈 Key Insights
- Sales increased during peak seasons
- Top products contributed majority of revenue
- Certain customer segments spend more

## ✅ Outcome
- Identified sales trends and customer behavior
- Provided business recommendations

## 🚀 Recommendations
- Focus on high-performing products
- Target valuable customers
- Improve low-performing categories
